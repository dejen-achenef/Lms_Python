## Usage Examples

```python
# Create metaverse campus
from lms_platform.apps.metaverse.models import MetaverseCampus

campus = MetaverseCampus.objects.create(
    name="Quantum Learning University",
    campus_type="space_station",
    environment_type="orbital",
    virtual_area=1000000,  # 1M square meters
    max_population=10000
)

# Join immersive classroom
classroom = campus.immersive_classrooms.get(
    room_type="zero_gravity"
)
```
